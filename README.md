
# Prompt System

This system helps you write a correct, clear, executable, and evaluable prompt for any project. The goal is not only to write a nice request, but to turn your idea into precise instructions that AI can understand and use to produce an output very close to what you want.

## System Components

### 1. `01-master-prompt-system.md`

The main system you use with any project.

### 2. `02-prompt-analysis-framework.md`

Prompt analysis framework: how AI analyzes your request before execution.

### 3. `03-perfect-prompt-template.md`

A ready-to-copy-and-fill prompt template.

### 4. `04-project-intake-questions.md`

Requirement-gathering questions before writing the prompt.

### 5. `05-100-score-rubric.md`

A 100-point prompt evaluation rubric.

### 6. `06-prompt-improver.md`

A prompt for improving any weak prompt and turning it into a strong version.

### 7. `07-output-control-rules.md`

Rules for controlling the response format, language, structure, depth, and quality.

### 8. `08-examples.md`

Practical examples of how to use the system.

## Quick Usage Method

1. Open `04-project-intake-questions.md` and answer the important questions.
2. Copy the template from `03-perfect-prompt-template.md`.
3. Fill in the template with your project information.
4. Use `05-100-score-rubric.md` to evaluate the prompt.
5. If the score is below 90, use `06-prompt-improver.md` to improve it.
6. When executing, add the control rules from `07-output-control-rules.md` as needed.

## Golden Rule

An excellent prompt does not just say:

> “Do this.”

An excellent prompt defines:

- The goal
- The context
- The audience
- The constraints
- The expected outputs
- The thinking method
- The success criteria
- What should be avoided

# نظام البرومبت 

هذا النظام يساعدك على كتابة برومبت صحيح، واضح، قابل للتنفيذ، وقابل للتقييم لأي مشروع. الهدف ليس فقط كتابة طلب جميل، بل تحويل فكرتك إلى تعليمات دقيقة يفهمها الذكاء الاصطناعي وينتج منها نتيجة قريبة جدًا مما تريده.

## مكونات النظام

1. `01-master-prompt-system.md`  
   النظام الرئيسي الذي تستخدمه مع أي مشروع.

2. `02-prompt-analysis-framework.md`  
   إطار تحليل البرومبت: كيف يحلل الذكاء الاصطناعي طلبك قبل التنفيذ.

3. `03-perfect-prompt-template.md`  
   قالب برومبت جاهز للنسخ والتعبئة.

4. `04-project-intake-questions.md`  
   أسئلة استخراج المتطلبات قبل كتابة البرومبت.

5. `05-100-score-rubric.md`  
   مقياس تقييم البرومبت من 100 نقطة.

6. `06-prompt-improver.md`  
   برومبت لتحسين أي برومبت ضعيف وتحويله إلى نسخة قوية.

7. `07-output-control-rules.md`  
   قواعد التحكم في شكل الإجابة، اللغة، التنسيق، العمق، والجودة.

8. `08-examples.md`  
   أمثلة عملية على استخدام النظام.

## طريقة الاستخدام السريعة

1. افتح ملف `04-project-intake-questions.md` وأجب عن الأسئلة المهمة.
2. انسخ قالب `03-perfect-prompt-template.md`.
3. عبئ القالب بمعلومات مشروعك.
4. استخدم ملف `05-100-score-rubric.md` لتقييم البرومبت.
5. لو النتيجة أقل من 90، استخدم `06-prompt-improver.md` لتحسينه.
6. عند التنفيذ، أضف قواعد التحكم من `07-output-control-rules.md` حسب الحاجة.

## قاعدة ذهبية

البرومبت الممتاز لا يقول فقط: "افعل كذا".  
البرومبت الممتاز يحدد: الهدف، السياق، الجمهور، القيود، المخرجات، طريقة التفكير، معايير النجاح، وما الذي يجب تجنبه.
