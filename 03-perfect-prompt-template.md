# 03 - قالب البرومبت المثالي

انسخ القالب التالي واملأ الفراغات.

---

## قالب عام لأي مشروع

أنت خبير في [اكتب المجال أو الدور المطلوب].

أريد منك مساعدتي في [اكتب المهمة الأساسية] بهدف [اكتب الهدف النهائي بوضوح].

## السياق

المشروع أو الفكرة هي:
[اشرح المشروع، الخلفية، الوضع الحالي، المشكلة، أو الفرصة]

## الجمهور المستهدف

الناتج موجه إلى:
[اكتب الجمهور: عملاء، فريق داخلي، مستثمرين، طلاب، مستخدمين، جمهور عام، إلخ]

مستوى الجمهور:
[مبتدئ / متوسط / خبير / غير تقني / إداري / تجاري]

## المطلوب بالتحديد

قم بالآتي:

1. [المهمة 1]
2. [المهمة 2]
3. [المهمة 3]
4. [أي مهمة إضافية]

## القيود والشروط

التزم بما يلي:

- اللغة: [العربية / الإنجليزية / الاثنين]
- النبرة: [رسمية / ودية / فاخرة / تسويقية / مباشرة / تعليمية]
- الطول: [قصير / متوسط / مفصل]
- التنسيق: [عناوين / جدول / نقاط / خطوات / ملف Markdown / كود / تقرير]
- لا تستخدم: [أشياء ممنوعة]
- يجب أن يتضمن: [أشياء ضرورية]

## طريقة العمل

قبل إخراج النتيجة النهائية:

1. حلل الهدف الحقيقي من الطلب.
2. حدد أي معلومات ناقصة.
3. اذكر افتراضاتك باختصار إذا كانت مهمة.
4. ابنِ الناتج بطريقة عملية قابلة للاستخدام.
5. راجع النتيجة مقابل معايير الجودة.

## شكل المخرج النهائي

أريد الناتج بهذا الشكل:

# [عنوان مناسب]

## 1. ملخص
[ملخص واضح]

## 2. التفاصيل
[الناتج الأساسي]

## 3. خطوات التنفيذ
[خطوات عملية]

## 4. ملاحظات الجودة
[نقاط مراجعة أو تحسين]

## معايير النجاح

اعتبر الإجابة ناجحة إذا كانت:

- واضحة جدًا
- قابلة للتنفيذ فورًا
- غير عامة أو سطحية
- منظمة
- مناسبة للجمهور
- ملتزمة بكل القيود
- لا تترك أسئلة مهمة بدون إجابة

إذا كان هناك غموض يمنع الوصول لنتيجة ممتازة، اسألني فقط عن أهم سؤالين قبل التنفيذ. أما إذا كان الغموض بسيطًا، افترض أفضل افتراض واذكره.
